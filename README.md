# SourceAddonsManager

This allows you to manage your own local copy of addons, outside the Workshop.

## Usage

### Client

1. Copy `configuration.example.json` to `configuration.json`
2. Fill out the information inside to fit your server.
3. Run `source.js`, and follow its instructions


### Server

1. Dedicate a folder just for this. For example, `http://10.0.44.20:5113/Mods/Left 4 Dead 2`
2. Download addons inside it. You can use [the helper scripts](./helper) to do it. For example, `./get.sh 3022416274` to download Left 4 Bots 2, and `./collection.sh 2066337798` to download the entire Glubtastic Collection
3. Make sure `data.json` exists. The helper script would do that for you
